Weekly Challenge

Removing duplicates from a sorted list.

You are given a pointer to the head of a sorted, single-linked list. The task is to remove any nodes with duplicate values, so that each element only appears once in the list.


Discuss this problem on Facebook.

Need a hint?

First think about how to iterate through the linked list. What algorithm could you use?

Once you can iterate through the list, try to find a way to check for duplicates. Remeber to watch out for the edges. It's easy to accidentally try and read an undefined/null value on the end of the list.

Once you can find duplicates, try to remove one without accidentally losing the rest of the list. Make sure that the previous node now points to the removed node's next. Remember, you can chain statements together to access furthers levels of depth, i.e. this.next.next = the next node of the next node.
